# Reel — 디자인 시스템

이 문서는 Reel 앱의 디자인 시스템을 정의한다.
다른 AI 툴(ChatGPT, Claude, Gemini 등)에서 화면을 디자인하거나 코드를 생성할 때 이 문서를 기준으로 삼는다.

---

## 앱 정체성

- **앱 이름:** Reel
- **슬로건:** 우리의 추억이 저절로 정리됩니다
- **컨셉:** 필름 릴(Film Reel) — 소중한 순간들을 차곡차곡 감아두는 것
- **타겟:** 전 연령 가족·커플·친구 (아기 부모 한정 아님)
- **핵심 기능:** AI 자동 이벤트 앨범 + 한국 인화(포토북) 연결

- 앱 이름: Reel (AI 추억 앨범 앱)
- 화면 크기: width 390px, 모바일 세로

색상:
- Primary: #1A1A2E (헤더, 다크 배경)
- Accent: #E94560 (버튼, 포인트)
- Background: #F8F8F8
- Text Primary: #1A1A2E
- Text Secondary: #8A8A9A

컴포넌트 스타일:
- 카드 모서리: 16~20px

포함할 요소: [여기에 요소 목록]

## 디자인 원칙

1. **추억이 주인공** — UI는 뒤로 물러나고, 사진·이벤트가 전면에 나온다.
2. **어둠과 밝음의 대비** — 헤더는 어둡게(#1A1A2E), 콘텐츠는 밝게(#F8F8F8). 사진이 빛나 보인다.
3. **따뜻한 포인트** — Accent(#E94560)는 감성적이고 따뜻한 레드-핑크. 차갑지 않다.
4. **가볍고 빠른 느낌** — frosted glass, 부드러운 그림자, 큰 모서리로 현대적이고 가볍다.
5. **한국어 우선** — 버튼·레이블은 한국어가 기본. 영어 병기는 브랜드명·고유명사만.
