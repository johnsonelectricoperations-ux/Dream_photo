# Dream Photo Roadmap

## Product Goal

모든 세대의 일상 사진을 AI가 자동으로 추억 앨범으로 만들고,
한국 인화와 연결해 손에 잡히는 기억으로 전달한다.

---

# Phase 0 — 방향 검증

## 경쟁 분석 + 포지셔닝 확정

기간: 2026 Q2

완료 조건:
- 경쟁 앱 분석 완료 (FamilyAlbum, Google Photos, Tidy, Gallery)
- 차별점 1줄 정의 완료
- 기술 가설 검증 (Gemini Vision API, ML Kit OCR)

상태: 완료

---

# Phase 1 — MVP 프로토타입

## AI 자동 이벤트 앨범

기간: 6~8주

핵심 기능:
- 갤러리 접근 + EXIF 읽기
- 시간·위치 기반 자동 이벤트 그룹화
- ML Kit 이미지 라벨링 (장면 인식)
- 이벤트 앨범 UI (월별 나열 아닌 이벤트 단위)
- 베스트컷 자동 선별

성공 기준:
- 여행에서 돌아왔을 때 앨범이 자동으로 만들어져 있음
- 사용자가 "이게 내가 찍은 사진 맞아?"라고 반응

---

# Phase 2 — 수익화 연결

## 포토북·인화 주문

기간: 4~6주

핵심 기능:
- 자동 생성 앨범 → 포토북 1클릭 주문
- 국내 인화 업체 API 연동 (스냅스·포토몬 등)
- 3일 배송

성공 기준:
- 첫 유료 포토북 주문 발생
- 재주문율 > 30%

---

# Phase 3 — 공유

## 가족·친구 공유

기간: 4주

핵심 기능:
- 앱 없이도 링크 하나로 앨범 열람
- 반응·댓글
- 50인 이상 공유 (FamilyAlbum 한계 극복)

성공 기준:
- 1명이 공유 → 2명 이상 신규 가입 (바이럴)

---

# Phase 4 — 전 연령 확장

## 아기 부모 외 타겟 확장

대상:
- 커플 (데이트 기록)
- 친구 그룹 (여행·모임)
- 반려동물 보호자
- 중년·노년 가족

---

# Phase 5 — 프리미엄 구독

## 수익 다각화

기능:
- 원본 화질 보존
- 자동 백업
- 베스트컷 AI 고급 선별
- 무제한 공유

수익 모델:
- 월 구독 4,900~9,900원

---

# Phase 6 — 글로벌

## 아시아 확장

한국 검증 후 일본·동남아 진출.
FamilyAlbum의 한계(일본 중심)를 역으로 공략.
