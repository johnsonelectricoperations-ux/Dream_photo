# Dream Photo

Dream Photo는 사진 관리 앱이 아니다.

사용자가 사진을 날짜나 파일명이 아닌
사람, 장소, 물건, 사건의 기억으로 찾을 수 있게 해주는
AI 기반 기억 검색 플랫폼이다.

## 문제

사람들은 사진을 날짜로 기억하지 않는다.

- 강릉 여행 사진
- 차번호 사진
- 불량품 사진
- 제품 모델명 사진

처럼 기억한다.

하지만 기존 사진 앱은 날짜와 파일 중심으로 저장한다.

## 비전

사진을 저장하는 앱이 아니라
기억을 검색하는 앱.

## MVP

- 휴대폰 사진 전체 분석
- AI 자동 태깅
- OCR 문자 추출
- 자연어 검색
- 이벤트(Event) 자동 생성
