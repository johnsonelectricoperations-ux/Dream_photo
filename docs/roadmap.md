# Dream Photo Roadmap

## Product Goal

사용자가 수천~수만 장의 사진 속에서
원하는 기억을 자연어로 검색할 수 있게 한다.

---

# Phase 0

## Problem Validation

기간

2026 Q2

목표

사용자가 실제로 어떤 사진을 찾고 싶어하는지 검증

완료 조건

- 사용자 인터뷰
- 실제 사례 수집
- 핵심 문제 정의

대표 사례

- 차번호
- 불량품
- 여행 사진

상태

완료 진행 중

---
