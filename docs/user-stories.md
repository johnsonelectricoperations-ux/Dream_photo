# User Stories

## Purpose

이 문서는 사용자가 Dream Photo를 어떻게 사용할 것인지 정의한다.

모든 기능은 실제 사용자 스토리에서 출발한다.

---

# MVP User Stories

## US-001

### 제목

사용자는 기존 사진을 한 번에 분석할 수 있다.

### 설명

앱 설치 후 사용자는 갤러리 접근 권한을 허용한다.

Dream Photo는 기존 사진들을 자동으로 분석한다.

### 성공 조건

- 사용자는 사진을 직접 하나씩 선택할 필요가 없다.
- 전체 사진 분석이 가능하다.

---

## US-002

### 제목

사용자는 사진 속 텍스트를 검색할 수 있다.

### 설명

Dream Photo는 OCR을 통해 사진 속 문자를 추출한다.

### 예시

사용자 검색:

차번호

또는

123가4567

### 성공 조건

관련 사진이 검색된다.

---

## US-003

### 제목

사용자는 장소로 사진을 검색할 수 있다.

### 설명

Dream Photo는 위치 정보와 AI 분석을 이용하여 장소를 추론한다.

### 예시

강릉

제주도

부산

### 성공 조건

관련 사진들이 검색된다.

---

## US-004

### 제목

사용자는 사물로 사진을 검색할 수 있다.

### 설명

AI는 사진 속 주요 객체를 분석한다.

### 예시

자동차

에어컨

명함

영수증

### 성공 조건

관련 사진이 검색된다.

---

## US-005

### 제목

사용자는 사건 기반으로 사진을 검색할 수 있다.

### 설명

Dream Photo는 여러 장의 사진을 하나의 기억(Event)으로 묶는다.

### 예시

강릉 여행

자동차 수리

제품 불량

### 성공 조건

관련 사진 그룹이 표시된다.

---

# Search Experience

## US-006

### 제목

사용자는 자연어로 검색할 수 있다.

### 설명

키워드가 아닌 일반 문장도 검색 가능해야 한다.

### 예시

강릉 갔을 때 찍은 사진

차번호 찍어둔 거

지난번 불량품 사진

### 성공 조건

의도에 맞는 결과를 보여준다.

---

# AI Analysis

## US-007

### 제목

앱은 사진을 자동 분석한다.

### 설명

사용자가 별도 태그를 입력하지 않아도 된다.

### AI 분석 항목

- 텍스트
- 장소
- 객체
- 날짜
- 이벤트 후보

### 성공 조건

자동 태그 생성

---

# Privacy

## US-008

### 제목

사용자는 자신의 사진이 외부로 전송되지 않는다고 신뢰할 수 있어야 한다.

### 설명

가능한 모든 분석은 기기 내에서 수행한다.

### 성공 조건

사용자가 개인정보 유출을 걱정하지 않는다.

# MVP Definition

MVP는 다음 기능을 포함한다.

- 기존 사진 전체 분석
- OCR
- 객체 인식
- 장소 인식
- 자연어 검색
- 이벤트 생성

MVP는 다음 기능을 포함하지 않는다.

- SNS 공유
- 친구 기능
- 영상 분석
- 클라우드 동기화
- AI 챗봇
