브라우저에서 PNG 추출 방법
1. logo.svg 파일을 브라우저에서 열기
2. 우클릭 → "다른 이름으로 저장" → SVG 저장
3. 무료 변환 도구 사용: svgtopng.com 에서 512×512 PNG 추출
4. 구글 플레이 콘솔 → 앱 아이콘에 512×512 PNG 업로드

또는 Inkscape(무료) 설치 후: inkscape logo.svg --export-png=icon_512.png -w 512 -h 512
